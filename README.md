En este respositorio se incluye un documento que incluye el análisis de la calidad del enlace de comunicación inalámbrica (WiFi 2.4GHz) entre dros drones. La versión definitiva en formato Lyx la tiene Carlos Calafate. Además, se incluye el proyecto Eclipse de la aplicación Java denominada "Dronning" utilizada durante los experimentos y de la cual sigue una ligera descripción.

Dronning es una aplicación desarrollada integramente en Java para Raspberry Pi. La aplicación, instalada en una Raspberry Pi situada sobre un dron, se conecta por el puerto serie con la controladora Pixhawk, recoge parámetros de vuelo y transmite esa información a otra Raspberry Pi. El segundo dispositivo recoge la información de vuelo de ambos drones y la registra en disco. Así mismo, también determina los paquetes de datos que se han perdido para así analizar la calidad del enlace de comunicación.
